# 校园二手书交易平台 - 需求规格说明书

## 1. 项目概述

### 1.1 项目背景
随着高校学生购书需求的增长，新书价格较高，二手书市场潜力巨大。建立一个专门的校园二手书交易平台，可以帮助学生以更低的价格获取所需教材和课外书籍，同时帮助有闲置书籍的学生变现。

### 1.2 项目目标
- 为在校学生提供一个安全、便捷的二手书交易渠道
- 支持书籍发布、浏览、搜索、购买、订单管理等功能
- 建立可信的交易机制，保护买卖双方权益
- 提供良好的用户体验和界面设计

### 1.3 目标用户
- 高校在校学生（本科、研究生）
- 即将毕业需要处理闲置书籍的毕业生

## 2. 功能需求

### 2.1 用户模块

#### 2.1.1 用户注册与登录
- 支持校园邮箱注册（限制学校域名）
- 支持手机号注册
- 登录方式：账号密码登录
- 找回密码功能（通过邮箱或手机验证码）

#### 2.1.2 用户信息管理
- 查看/修改个人资料（昵称、头像、院系、年级）
- 查看个人账户信息
- 收货地址管理（新增、编辑、删除、默认地址设置）

### 2.2 书籍模块

#### 2.2.1 书籍发布
- 填写书籍信息：书名、作者、ISBN、出版社、版本
- 设置书籍分类（教材类、考研资料、课外阅读、其他）
- 上传书籍图片（最多5张）
- 填写书籍详情（成色、新旧程度、使用痕迹）
- 设置价格（原价参考价、售价）
- 填写交易方式（自提、快递）
- 设定库存数量

#### 2.2.2 书籍浏览与搜索
- 首页推荐热门/最新书籍
- 分类浏览（按分类、院系）
- 关键词搜索（书名、作者、ISBN）
- 筛选功能（价格区间、成色、交易方式）
- 排序功能（最新发布、价格从低到高、价格从高到低）

#### 2.2.3 书籍详情
- 查看书籍详细信息
- 查看卖家信息
- 查看书籍图片轮播
- 加入购物车
- 立即购买
- 联系卖家（站内消息）

### 2.3 购物车模块
- 添加书籍到购物车
- 修改购买数量
- 删除购物车商品
- 批量下单
- 查看购物车总价

### 2.4 订单模块

#### 2.4.1 创建订单
- 选择收货地址
- 选择支付方式（暂只支持线下支付/余额支付）
- 提交订单

#### 2.4.2 订单管理
- 查看所有订单（全部、待付款、待发货、待收货、已完成、已取消）
- 取消订单（未付款状态下）
- 确认收货
- 申请退款/退货
- 查看订单详情

#### 2.4.3 订单状态流转

### 2.5 消息通知模块
- 系统通知（活动、公告）
- 订单状态变更通知
- 买家咨询通知
- 消息已读/未读状态

### 2.6 评价模块
- 交易完成后可对商品进行评价
- 评价内容：评分（1-5星）、文字评价
- 查看卖家信誉评分

---

## 3. 非功能需求

### 3.1 性能需求
- 页面加载时间不超过3秒
- 搜索响应时间不超过1秒
- 支持高并发访问

### 3.2 安全需求
- 用户密码加密存储
- 敏感信息脱敏处理
- 防止SQL注入和XSS攻击

### 3.3 界面设计
- 简洁清晰的界面设计
- 移动端响应式布局
- 统一的视觉风格

---

## 4. 技术架构

### 4.1 前端
- Vue 3
- UI组件库：Element Plus
- 状态管理：Pinia / Vuex

### 4.2 后端
- Python Flask + Flask-SQLAlchemy (ORM)
- 数据库：SQLite 3
- 缓存：Redis
